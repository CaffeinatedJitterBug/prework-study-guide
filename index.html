HTML

  • HTML code is normally contained in a file called "index.html" which makes the main html file easy to locate and streamlines the publishing process.
  • Everything on a web page is defined by HTML elements.
  • The head element contains information about the webpage.
  • The meta element sets meta data needed to structure the site, such as charset to set what chaaracters are being used, and information on how to format the page.
  • The link element can be used to link a CSS sheet, this link should be put in the header before the title and after any meta data. Be sure to include rel="stylesheet" href=(link)
  • The title element sets what is displayed in the tab title
  • The body element represents the visible content shown to the user.
  • The body element is where you put things like headers, sections, lists, images and links.
  • Unordered lists are bullet points, ordered lists are numbered lists.
  • The header types are h1-h6.
  • The script element can be used to link a JavaScript file, so we can use Javascript to make our website interactable. src sets the location for our Javascript file. For example, this web page's Javascript src is set to "./assets/script.js" "." representing the relative directory our index.html file is in.

CSS

  • CSS can be set up with either inline CSS, an internal CSS style sheet, or an external CSS style sheet, though the external style sheet is the most common method.
  • CSS elements should be structured with the name of the element to be styled, followed by {} containing style elements.
  • Multiple elements can be styled in one block by listing them with commas seperating them. Ex: header, footer
  • The size of CSS elements are measured in px (pixels)
  • Height and width determine the size of an element.
  • Display sets how an element is displayed. For example, in our style sheet, we have our images set to display: block by default, which puts any images on our web page on a new line and fills the entire width.
  • A margin indicates how much space we want around the outside of an element.
  • Margins can be defined by either specifying individual margins such as margin-left or margin-right, or it can be defined simply with margin: (x px) (y px)
  • Padding indicates how much space we want around the content inside an element.
  • A box shadow (box-shadow) makes a shadow around an element, taking a x offset and y offset based off of the position of the element, as arguments, as well as color.

Git

  • code . : Starts VSCode
  • git status: checks what branch we are currently on
  • git checout main: returns to the main branch. Answer 'n' when branch deletion fails
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add -A: commits all files in the working branch
  • git commit -m "notes": commits all changes to the repository with a message attached
  • git pull origin main: pulls the main branch into the feature branch to ensure we're working on the most recent version. Always pull before making or pushing any changes
  • git push origin (branch name): updates main branch from specified branch. Remember to get the push approved on GitHub

Javascript

  • Javascript is a versatile programming language used for a lot of different things, such as APIs or frameworks and libraries for HTML.
  • Variables are containers that hold a value, such as a string or number. They are set with var (variableName) = value
  • console.log() logs whatever is contained in the parentheses to the console. Can contain a value such as a string, or it can contain a variable name, which will log whatever value is stored in the variable when the console.log is called.
  • If statements takes a parameter and compares it against a list of conditions to decide what code to run.
  • A for loop takes a set of parameters and runs the code it contains as long the conditions set in the parameters are true.
  • A function is a block of code that can be called any time after the fuction is defined. It helps to reduce repeatitive code.
